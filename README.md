# cfs-service-api
Call for Service API Testing 
### Project Setup
	Java JDK 11 
	Maven
	MongoDB
	MongoCompass
	Lombok - prefer in the Lombok folder in Git repo.
	
### Run
	-> Project folder run :
	mvn clean install
	mvn spring-boot-run
### Testing
	- PostMan
	- SwaggerUI $host/api/v1/swagger-ui.html
